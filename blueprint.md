# Lotto Number Generator

## Overview

A simple web application to generate lottery numbers.

## Features

*   **Lottery Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **User Interface:** A clean and simple interface to display the generated numbers.
*   **Theme Toggle:** Switch between light and dark mode.
